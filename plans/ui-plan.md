# Web UI Plan — Elect Righteous

## Design Vision

A faith-themed, professional election intelligence platform inspired by the design language of celebratejesus.org. The UI should feel trustworthy, authoritative, and accessible — like a well-designed church website meets a civic research tool.

## Tech Stack

- **Framework**: Next.js 15+ (static export)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion (subtle, professional)
- **Icons**: Lucide React
- **Deployment**: Cloudflare Pages
- **Data**: Static JSON generated by Report Generator agent

```css

font-family: 'Playfair Display', Georgia, serif;

/* Body: Clean sans-serif for readability */

```

### Component Library

1. **Hero Section** — Full-width, background image with dark overlay, centered text
2. **Location Search** — Prominent search bar with autocomplete for US addresses
3. **Election Card** — Card displaying an upcoming election with date, positions, candidate count
4. **Candidate Card** — Photo, name, party, position, consistency rating badge
5. **Comparison Table** — Side-by-side candidate comparison with values alignment
6. **Quote Block** — Styled verbatim quote with source citation
7. **Values Badge** — Color-coded alignment indicator (Strong/Moderate/Weak/Unknown)
8. **Source Citation** — Inline source link with hover preview
9. **Red/Green Flag Alert** — Highlighted concern or positive finding
10. **Report Section** — Collapsible section for detailed analysis

## Pages

### 1. Home Page (`/`)
- Hero with mission statement and scripture
- Location search bar (prominent, center stage)
- "Featured Elections" section (current active research)
- "How This Works" section (methodology transparency)
- "Our Values Framework" section (link to full framework)

### 2. Election Page (`/elections/[slug]`)
- Election details (date, positions, jurisdiction)
- All candidates in the race
- Side-by-side comparison
- Key issues comparison
- Values alignment comparison

### 3. Candidate Page (`/candidates/[slug]`)
- Full dossier rendered beautifully
- Collapsible sections for each analysis area
- Timeline of key quotes and positions
- Campaign finance visualization
- Source list (every source cited, clickable)

### 4. About Page (`/about`)
- Mission statement
- Methodology explanation
- Values framework overview
- Team/project info
- Disclaimer (non-partisan, evidence-based)

### 5. Search Results (`/search`)
- Results by location, candidate name, or election
- Filter by state, election type, date

## Mobile Design
